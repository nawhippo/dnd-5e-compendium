## Installation

### Method 1: Manifest URL
1. In Foundry VTT, go to the "Add-on Modules" tab
2. Click "Install Module"
3. Paste this manifest URL: `https://github.com/nawhippo/dnd-5e-compendium/releases/latest/download/module.json`
4. Click "Install"

### Method 2: Manual Installation
1. Download the latest release
2. Extract to your Foundry VTT `Data/modules` folder
3. Restart Foundry VTT
4. Enable the module in your world

## Usage

1. Enable the module in your game world
2. The compendium packs will appear in your compendiums sidebar:
   - "Player's Handbook Subclasses"
   - "Player's Handbook Class Features"
3. Drag and drop subclasses into character sheets
4. All subclass features are included in the descriptions

## Compatibility

- **Foundry VTT**: Version 11+
- **D&D 5e System**: Version 2.0.0+
- Tested and verified with Foundry VTT v11 and v12
